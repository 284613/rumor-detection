# 社交媒体恶意谣言识别系统

## 项目介绍（200字）

随着社交媒体的快速发展，网络谣言传播速度呈几何级增长，对社会稳定、公共安全和个人权益造成严重威胁。本项目旨在研究与实现基于深度学习的谣言检测方法，通过自然语言处理技术自动识别社交媒体平台上的虚假信息，为遏制谣言传播提供技术支撑。主要研究目标包括：构建高效的谣言检测模型，提升模型在中文社交场景下的识别准确率，开发可演示的原型系统，为相关领域的研究与应用提供参考。

## 技术方案（300字）

本系统采用"预训练模型+特征提取+分类器"的整体架构。首先使用BERT预训练模型对输入文本进行编码，获取深层语义表示；接着通过特征提取模块对BERT输出进行处理，提取文本的关键特征；最后使用全连接分类器完成真实信息与谣言的二分类任务。核心技术包括：BERT-base-chinese预训练模型，用于捕捉中文文本的上下文语义信息；多头注意力机制，用于聚焦文本中的关键内容；交叉熵损失函数，用于优化模型参数。创新点在于：针对中文社交文本特点优化了输入预处理流程，结合了词性特征与语义特征，设计了轻量化模型结构以满足实时检测需求。

## 数据集（200字）

本项目主要使用Weibo-2016和Weibo-2018数据集，这两个数据集是中文谣言检测领域的权威基准数据集。Weibo-2016包含约3,000条微博数据，其中谣言与真实信息各占约50%；Weibo-2018数据集规模更大，包含约4,500条标注数据，涵盖更多样化的谣言类型。数据格式为JSON/CSV，每条数据包含以下字段：`id`（微博唯一标识）、`text`（微博文本内容）、`label`（标注标签，0表示真实信息，1表示谣言）、`time`（发布时间）、`comments`（评论列表，可选）。数据集按8:1:1的比例划分为训练集、验证集和测试集。

## 环境配置（200字）

**硬件要求**：推荐配置为Intel i5及以上处理器、16GB及以上内存、NVIDIA GPU（显存8GB及以上，用于加速模型训练）；最低配置为Intel i3处理器、8GB内存，可使用CPU进行推理但训练速度较慢。

**软件依赖**：Python 3.9及以上版本，主要依赖库包括PyTorch 1.12+、Transformers 4.25+、Datasets 2.8+、Pandas 1.3+、NumPy 1.21+、Scikit-learn 1.0+、Jieba 0.42+。

**安装步骤**：1. 克隆项目代码至本地；2. 创建Python虚拟环境（推荐使用conda）；3. 执行`pip install -r requirements.txt`安装依赖；4. 下载预训练模型权重至指定目录。

## 运行说明

### 训练命令示例
```bash
# 使用默认参数训练
python train.py

# 指定参数训练
python train.py --batch_size 16 --epochs 5 --lr 2e-5
```

### 评估命令示例
```bash
# 评估模型性能
python evaluate.py --checkpoint ./models/best_model.pth

# 输出详细评估指标
python evaluate.py --checkpoint ./models/best_model.pth --detailed
```

### 预期输出
- 训练过程中会输出每个epoch的训练损失、验证损失和准确率
- 评估完成后会输出准确率、精确率、召回率和F1分数等指标
- 最佳模型权重会保存至`./models/`目录下
