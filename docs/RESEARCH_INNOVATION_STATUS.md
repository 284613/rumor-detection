# 谣言检测课题：创新点与研究现状 (Research Innovation & Status)

本页汇总了本课题的核心创新点与研究现状描述，可直接用于中期报告、开题报告或论文撰写。

## 一、 完整创新点表述 (Innovative Points)
**填入“创新特色”栏：**

**创新点名称**： 面向极早期冷启动场景的多立场约束 LLM 传播树增强与谣言预警机制

**正文内容**：
针对社交媒体谣言爆发初期传播节点极度稀疏、现有图神经网络模型（如 Bi-GCN、RvNN）因缺乏有效拓扑结构而出现"冷启动"失效问题，本课题在已有 LLM 虚拟节点增强研究（LLM-VN，2026）的基础上，提出了一种更具针对性的多立场约束 LLM 局部路径增强机制。

该方法的核心区别与创新体现在三个层面：
1. **场景针对性**：专门面向极早期场景（传播深度 ≤ 2 层，真实回复数 ≤ 5 条）设计数据截断与增强流程，打破了传统模型被动等待传播树成型的限制；
2. **生成约束性**：在 LLM 生成环节引入"多立场强制约束"Prompt 工程，要求模型同时输出支持、反对与中立三类预测评论，有效规避了因 LLM 幻觉偏向而导致的"立场污染"问题，保证了虚拟节点对真实舆论生态的多样性模拟；
3. **架构融合性**：将增强后的多关系传播图谱接入谣言分类与立场检测的联合多任务学习框架，使极早期特征稀疏问题得到结构层面的弥补，最终实现从"被动观测"向"主动演化预测"的转变。

与 LLM-VN 等同期工作相比，本方案的差异化贡献在于：明确界定极早期冷启动为核心目标场景，并提出可验证的多立场生成约束策略，辅以消融实验验证增强节点中立场多样性对早期预警准确率的实质影响。

## 二、 研究现状与发展趋势 (Research Status)
**填入“研究现状和发展趋势”栏：**

近年来，社交媒体谣言检测经历了从浅层机器学习向图神经网络（Bi-GCN、RvNN、PLAN 等）的演进。然而，此类模型高度依赖完整的级联传播结构，在谣言传播极早期面临严重的特征稀疏与冷启动瓶颈。

随着大语言模型的兴起，2024—2026 年的前沿研究开始将 LLM 引入谣言检测的三条技术路线：
1. **直接分类**：将 LLM 直接用于分类判断（Zeng et al., 2025；SePro），但研究发现其在长结构化上下文下性能下降；
2. **辅助推理**：以 LLM 作为"顾问"辅助小模型蒸馏推理（Hu et al., 2024）；
3. **数据增强与补全**：利用 LLM 生成能力进行数据增强与传播拓扑补全（RumorLLM, 2024；LLM-VN, 2026）。

本课题紧跟第三条路线，在其基础上进一步聚焦极早期冷启动场景与多立场生成约束这一尚待深入探索的研究问题，以满足网络治理中"打早打小"的现实需求。
