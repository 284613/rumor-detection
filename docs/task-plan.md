# 谣言检测毕业设计任务清单

## 一、实验环境搭建

### P0级任务（核心必要任务）

#### 任务1: 硬件环境准备
- **描述**: 准备满足要求的硬件设备
- **详细内容**:
  - GPU: RTX 3060/4090或Tesla系列，显存≥8GB
  - 内存: ≥16GB
- **预计耗时**: 1-3天（根据设备 availability）
- **依赖关系**: 无
- **待确认事项**: 具体硬件型号确认

#### 任务2: 操作系统安装
- **描述**: 安装Ubuntu操作系统
- **详细内容**:
  - 操作系统版本: Ubuntu 20.04 LTS 或 22.04 LTS
- **预计耗时**: 2-4小时
- **依赖关系**: 任务1完成
- **待确认事项**: 具体操作系统版本选择

#### 任务3: 基础软件环境配置
- **描述**: 安装Anaconda/Miniconda和Python环境
- **详细内容**:
  - 安装Anaconda或Miniconda
  - 创建Python 3.9虚拟环境
- **预计耗时**: 1-2小时
- **依赖关系**: 任务2完成
- **待确认事项**: 无

#### 任务4: 项目目录结构创建
- **描述**: 创建项目所需的目录结构
- **详细内容**:
  - 创建目录: /data, /models, /app, /utils
  - 创建文件: /train.py, /evaluate.py, requirements.txt
- **预计耗时**: 30分钟
- **依赖关系**: 任务3完成
- **待确认事项**: 无

#### 任务5: 深度学习框架与依赖包安装
- **描述**: 安装PyTorch及所有依赖库
- **详细内容**:
  - PyTorch（根据CUDA版本选择对应安装命令）
  - HuggingFace transformers/datasets
  - scikit-learn, pandas, numpy
  - matplotlib, seaborn
  - jieba, requests, beautifulsoup4
- **预计耗时**: 1-2小时
- **依赖关系**: 任务4完成
- **待确认事项**: PyTorch与CUDA版本匹配确认

---

## 二、训练数据采集与构建

### P0级任务（核心必要任务）

#### 任务6: 开源基准数据集下载
- **描述**: 下载并整理开源数据集
- **详细内容**:
  - Weibo-2016/2018微博谣言数据集
  - Twitter数据集(PHEME)
- **预计耗时**: 2-4小时（取决于网络速度）
- **依赖关系**: 任务5完成
- **待确认事项**: 数据集下载链接待补充（已在 dataset-resources.md 中）

#### 任务7: 数据集预处理
- **描述**: 对下载的数据集进行清洗和预处理
- **详细内容**:
  - 数据格式统一
  - 缺失值处理
  - 数据格式转换为模型可读取格式
- **预计耗时**: 3-5小时
- **依赖关系**: 任务6完成
- **待确认事项**: 无

### P1级任务（重要任务）

#### 任务8: 自定义爬虫开发
- **描述**: 开发微博、知乎等平台的数据爬虫
- **详细内容**:
  - 热门话题数据采集
  - 辟谣板块数据采集
  - 反爬虫策略应对
- **预计耗时**: 3-5天
- **依赖关系**: 任务7完成
- **待确认事项**: 具体采集平台和范围确认

#### 任务9: 数据增强实现
- **描述**: 实现数据增强方法
- **详细内容**:
  - LLM文本重写
  - 立场改写
- **预计耗时**: 2-3天
- **依赖关系**: 任务8完成
- **待确认事项**: 使用的LLM模型确认

#### 任务10: 传播树构建
- **描述**: 构建谣言传播树结构
- **详细内容**:
  - 记录status_id和parent_id
  - 使用NetworkX构建树结构
- **预计耗时**: 2-3天
- **依赖关系**: 任务9完成
- **待确认事项**: 无

### P2级任务（锦上添花任务）

#### 任务11: 标注质量控制流程搭建
- **描述**: 建立数据标注质量控制机制
- **详细内容**:
  - 3人背靠背标注流程
  - Kappa系数计算工具
- **预计耗时**: 1-2天
- **依赖关系**: 任务10完成
- **待确认事项**: 标注人员安排确认
